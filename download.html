System Requirements

Digital Terrarium is designed to run smoothly on a wide range of hardware. Here's what you need:

  • 4 GB RAM minimum (8 GB recommended)
  • 500 MB available storage space
  • Graphics card with OpenGL 2.0 support or better
  • 1280x720 minimum screen resolution

What's Included

Every download includes everything you need to start creating:

  • Full desktop application with 3D rendering engine
  • Asset import support for PNG, JPG, and GLB files
  • Pre-configured terrarium environments
  • Built-in behavior system for autonomous movement
  • Local file storage with no cloud requirements

Getting Started

Once you've downloaded and installed Digital Terrarium, you're ready to go:

  • Import your first asset by dragging in an image or 3D model
  • Place it in a terrarium and watch it come to life
  • Assign behaviors to add movement and personality
  • Save your creation and return to it anytime

No tutorials required. No steep learning curves. Just pure creative joy.
